### UTM Tracking Template
```
?utm_source={platform}&utm_medium={medium}&utm_campaign={campaign}&utm_content={ad_name}
```

Example:
```
https://shivkathiawadithali.com/?utm_source=instagram&utm_medium=reel&utm_campaign=awareness_jan26&utm_content=thali_video_v1
```
