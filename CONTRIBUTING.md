# Contributing to MetroMate — Shiv Kathiawadi Thali Brand Development

Thank you for your interest in contributing to this project!

## How to Contribute

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes** following the style guidelines below
4. **Commit**: `git commit -m "feat: describe your change"`
5. **Push**: `git push origin feature/your-feature-name`
6. **Open a Pull Request**

## Commit Message Convention

We follow [Conventional Commits](https://www.conventionalcommits.org/):

```
feat:     New feature or strategy addition
fix:      Correction or update to existing content
docs:     Documentation updates
content:  New marketing content or assets
data:     Analytics or reporting updates
```
