# 🔍 SEO & Local Visibility Checklist

## Google Business Profile

- [ ] Claim and verify Google Business Profile
- [ ] Add complete business name, address, phone (NAP)
- [ ] Set accurate business category: "Gujarati Restaurant" / "Indian Restaurant"
- [ ] Upload 10+ high-quality photos (interior, food, team)
- [ ] Add menu with pricing
- [ ] Enable Google Messaging
- [ ] Set business hours (including holiday hours)
- [ ] Write keyword-rich business description (750 chars)

## Review Strategy

- [ ] Request reviews from every satisfied customer via WhatsApp link
- [ ] Respond to all reviews within 24 hours
- [ ] Target: 50 reviews in first 2 months
- [ ] 4.5+ star average goal

## On-Page SEO (Website)

- [ ] Title tag: "Shiv Kathiawadi Thali | Authentic Gujarati Food in Surat"
- [ ] Meta description with location keywords
- [ ] H1 with primary keyword
- [ ] Schema markup: `LocalBusiness`, `Restaurant`, `Menu`
- [ ] Mobile-first design
- [ ] Page load speed < 3 seconds
- [ ] Image alt tags with keywords
